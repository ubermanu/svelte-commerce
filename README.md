# Svelte Commerce

Svelte Commerce is a fully open source, headless, e-commerce platform built with SvelteKit and Magento 2.

## Features

- SvelteKit
- Magento 2 GraphQL
- Melt UI
- Tailwind CSS
- TypeScript

## GraphQL

Retrieve the schema and types from your Magento 2 instance by running the following command:

```bash
pnpm run codegen
```
